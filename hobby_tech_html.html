HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure of a webpage, using elements represented by tags enclosed in angle brackets (e.g., <tag>).

Tips and Tricks

  • Use semantic tags like <header>, <footer>, <article>, and <section> for better readability and SEO.
  • Always include the declaration at the top of your document to ensure proper rendering.
  • Use comments (<!-- Comment here -->) to leave notes in your code.

Examples of Common Tags

Tag Description
<h1>...</h1> Defines the largest heading.
<p>...</p> Defines a paragraph.
<a href="url">...</a> Defines a hyperlink.
<img src="url" alt="description"> Embeds an image.
<table>...</table> Creates a table.

HTML Content Examples

Basic Structure

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Page Title</title>
                    </head>
                    <body>
                        <h1>This is a Heading</h1>
                        <p>This is a paragraph.</p>
                    </body>
                </html>

Table Example

                <table>
                    <tr>
                        <th>Name</th>
                        <th>Age</th>
                    </tr>
                    <tr>
                        <td>Alice</td>
                        <td>25</td>
                    </tr>
                    <tr>
                        <td>Bob</td>
                        <td>30</td>
                    </tr>
                </table>

Form Example

                <form>
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name">
                    <br>
                    <input type="submit" value="Submit">
                </form>
